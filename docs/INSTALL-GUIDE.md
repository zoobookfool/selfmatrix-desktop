## SelfMatrix って何?

SelfMatrix は、Matrix というオープンなチャットの仕組みの上に作った、Discord に近い使い心地の
チャット + ボイスチャットです。友達サークルの中で使うことを想定しています。

使い方は 2 通りあります。

- **Windows アプリ (native)**: Windows パソコンに専用アプリをインストールして使う。通話まわりの
  使い勝手が一番良い版です。
- **ブラウザで使う (web)**: パソコン/スマホ/Mac のブラウザで URL を開くだけ。アプリのインストールが
  要らない、どの端末でも使える版です。

**どちらを使っても同じ会話・同じ通話に参加できます。** Windows アプリの友達とブラウザの友達が、
同じテキストチャンネル・同じボイスチャンネルで一緒に話せます (詳しくは後述)。

Windows アプリを入れられる/入れたい友達は「1. Windows アプリのインストール」へ、
それ以外 (スマホ、Mac、あるいは単に インストールが面倒な人) は「2. ブラウザで参加する」へ進んでください。

### 1.2 「発行元不明」の警告が出たら (Windows SmartScreen)

このアプリは個人開発のため、Windows が求める「コード署名」という有料の認証を付けていません。
そのため、初めて起動するときに Windows から次のような警告が出ることが**想定されています**
(悪いソフトという意味ではありません)。

> **Windows によって PC が保護されました**
> Microsoft Defender SmartScreen は認識されないアプリの起動を停止しました。このアプリを実行す
> ると、PC が危険にさらされる可能性があります。

これが出たら、慌てずに以下の手順で進めてください。

1. 警告ウィンドウの中にある **「詳細情報」** という小さいリンクをクリックする。
2. 「実行」ボタンが出てくるので、それをクリックする。

これで通常どおりインストールが始まります。

**注意**: 見知らぬ人から届いたリンクで同じ警告が出た場合は簡単に信用しないでください。
今回は「SelfMatrix の運用者から直接案内されたダウンロード先」からダウンロードしていることが
前提です。不安な場合は、次の「1.3 ダウンロードしたファイルの確認 (任意)」を行うか、
運用者に直接確認してください。

### 1.3 ダウンロードしたファイルの確認 (任意・やりたい人向け)

「本当に運用者が公開したファイルそのものか」をもう少し厳密に確かめたい人向けの、任意の手順です。
面倒であれば飛ばして 1.4 に進んで問題ありません。

以下のうち、どちらか (できれば両方) を確認すると安心です。

**方法 A: SHA256 のハッシュ値を照合する**

ダウンロードした `.exe` ファイルのハッシュ値 (ファイルの「指紋」のようなもの) が、
GitHub の Releases ページに置いてある `SHA256SUMS` というファイルの値、および
運用者が別経路 (Matrix の運用ルームなど) で案内する値と一致するか確認します。
両方が一致すれば、ダウンロード経路の改ざんが無いことの裏付けになります。

```powershell
# PowerShell で実行 (ファイル名は実際にダウンロードしたものに置き換える)
Get-FileHash .\SelfMatrix-Setup-X.Y.Z.exe -Algorithm SHA256
```

表示された値と、`SHA256SUMS` ファイル内の値、および運用ルームに掲示された値を見比べてください。

**方法 B: GitHub の「ビルド証明 (Artifact Attestation)」を確認する**

「このファイルが、公開されているソースコードから、正規のビルド手順で作られたものである」ことを
検証できます。GitHub の `gh` コマンドラインツールが必要で、GitHub アカウントでのログインが必要です
(匿名では確認できません)。

```sh

```

### 1.4 初回起動とアカウント作成 (matrix.org)

インストールが終わるとアプリが起動します。初めて開くと、接続先のサーバー (homeserver) を
選ぶ画面が出ます。

- **候補に出ている「matrix.org」(Matrix 公式の無料サーバー) をそのまま使ってください。**
  サーバーの URL を手入力する必要はありません。私たちのサークルは Matrix の連合 (federation)
  でつながるので、あなたのアカウントは matrix.org 上に作れば OK です。
- 「アカウントを作成」を選び、ユーザー名とパスワードを自分で決めて登録してください
  (matrix.org のアカウントなので、あなた自身のものです)。
- **すでに matrix.org や他の Matrix サーバーのアカウントを持っている人は、それでログインするだけで
  かまいません** (新規に作る必要はありません)。

> 補足: 「別のサーバーを使う」で自分の Matrix サーバーを持っている人はそれを使ってもかまいません。
> 通話も連合越しに一緒に成立します。

### 1.5 ログインと、サークルへの招待

1. 上で作った (または既存の) アカウントでログインします。
2. **ログイン後、運用者からの「ルームへの招待」が届きます** (届いていなければ、あなたの
   ユーザー名 — 例 `@yourname:matrix.org` — を運用者に伝えてください。招待を送ります)。
3. 招待を承認すると、サークルのテキストチャンネル・ボイスチャンネルに参加できます。

**ユーザー名・パスワードは他人に教えないでください。** このガイド自体にも実際のアカウント情報は
書きません。

### 1.6 新しいデバイスの認証 (暗号化された会話を読むために)

SelfMatrix の会話の多くは暗号化 (E2EE) されています。安全性を保つ仕組みとして、
**新しいデバイスでログインしたときに「このデバイスを認証する」手順が必要**になります
(cinny の画面ではこの手順は「デバイス認証」または「手動で認証」と呼ばれています)。

大まかな流れは次のとおりです (実際の画面は `[TODO]` として下に画像を追加予定)。

1. ログイン後、「このデバイスを認証してください」といった案内が出ます。
2. 認証方法は主に 2 通りです。
   - **他の認証済みデバイスから承認する**: すでにログイン済みの自分の別端末 (スマホ版 web など)
     がある場合、そちらに表示される絵文字と、新しいデバイス側に表示される絵文字が
     同じ並び順であることを確認して「一致する」を押す。
   - **リカバリーキーで手動認証する**: 自分の他デバイスが無い場合、**リカバリーキー**
     (最初にアカウントを暗号化設定したときに発行される、長い文字列またはファイル) を
     入力して認証する。
3. あなたが**このサークルで一番最初にログインする人**の場合、逆にこの端末が
   「デバイス認証のセットアップ」を求められ、**リカバリーキー (と、任意でパスフレーズ) が
   新規発行**されます。表示された・ダウンロードしたリカバリーキーは、
   **なくすと過去の暗号化メッセージが読めなくなる**ため、パスワードマネージャーなど
   安全な場所に保管してください。人に見せたり送ったりしないでください。

### 1.7 トレイ常駐について

SelfMatrix の Windows アプリは、ウィンドウを**閉じるボタン (×) を押してもアプリ自体は終了せず、
タスクトレイ (画面右下の通知領域) に常駐し続けます** (Discord と同じ動作です)。これは
「通話中に誤ってウィンドウを閉じて通話が切れる」ことを防ぐための仕様です。

- ウィンドウを閉じても通知は届き続けます。
- アプリを完全に終了したいときは、タスクトレイのアイコンを右クリックして出るメニューから
  「終了」を選んでください。

## 3. Windows アプリの友達とブラウザの友達は同じ通話に混ざれます

これは重要なポイントなので明記しておきます。

**Windows アプリ (native) を使っている友達と、ブラウザ (web) を使っている友達は、
同じテキストチャンネル・同じボイスチャンネルに一緒に参加できます。** 中身は同じ Matrix
サーバー・同じ通話基盤 (Element Call) を使っているため、見た目や一部の使い勝手 (通話の
ウィンドウの出し入れなど) が違うだけで、会話や通話そのものは完全に混ざって成立します。

- 誰か 1 人が Windows アプリで通話を始めて、別の友達がブラウザから同じボイスチャンネルに
  入って合流する、といった使い方も普通にできます。
- Windows アプリ側だけの機能 (通話ウィンドウの切り離し、システム音声つきの配信など) は
  ありますが、通話に参加すること自体には影響しません。

**通話は「ホスト (運用者) が先に入ってから」参加してください。** 通話の音声・映像は、その通話に
最初に入った人のサーバー設備を使う仕組みになっています。matrix.org などサークル外のアカウントは
自前の通話設備を持たないため、**サークルのホスト (運用者) が先にボイスチャンネルに入り、その後で
みんなが合流する**とうまくいきます。入ったのに相手の声が聞こえない・つながらないときは、一度
退出して、ホストが入っているのを確認してから入り直してみてください。

## 4. 困ったときは (FAQ)

**Q. SmartScreen の警告で「実行」ボタンが見当たらない/先に進めない**
A. 警告ウィンドウの中の小さい文字「詳細情報」をクリックすると、その下に「実行」ボタンが
現れます。ボタンが最初から大きく表示されているわけではないので見落としやすいです。
それでも進めない場合は運用者に相談してください。

**Q. ログインできない (パスワードが違う、ユーザー名が見つからないと言われる)**
A. 招待時に案内されたユーザー名・パスワードを再確認してください。それでも解決しない場合は
運用者に連絡してください (`[TODO: 連絡手段を記入。例: 招待時に案内する別の連絡先]`)。

**Q. 通話に入ったのに音や映像がつながらない**
A. よくある原因は次のとおりです。
- ブラウザ/アプリからマイク・カメラへのアクセス許可を求められたときに「許可」を選んだか確認する。
- Windows アプリの場合、Windows の「設定 → プライバシー → マイク/カメラ」でアプリ全般の
  アクセスがブロックされていないか確認する。
- 何度か再読み込み/再起動してみる。
それでも解決しない場合は運用者に連絡してください。

**Q. 「デバイス認証」の画面で詰まった。リカバリーキーが分からない**
A. リカバリーキーは、あなたが最初にアカウントの暗号化を設定したときに一度だけ表示・
発行されるものです。心当たりの保存場所 (パスワードマネージャー、メモ、ダウンロード
フォルダなど) を確認してください。見つからない場合は運用者に相談してください
(過去の暗号化メッセージが読めなくなる可能性があります)。
